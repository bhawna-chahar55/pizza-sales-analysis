                                          📊 Pizza Sales Analysis (SQL)
📖 Overview

This project analyzes pizza sales data using SQL to understand sales performance, customer behavior, and product demand.

🎯 What I Did

Calculated total revenue

Found top-selling and most profitable pizzas

Analyzed sales by category

Identified peak order hours

Used window functions to find top pizzas in each category

📊 Key Insights

A few pizzas generate most of the revenue

Orders are higher during specific hours of the day

Some categories perform better than others

🛠️ Skills Used

SQL, Joins, Aggregations, Subqueries, Window Functions

🚀 Why This Matters

These insights can help improve menu decisions, pricing, and business planning.
